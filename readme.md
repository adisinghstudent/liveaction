# ScreenKnow - AI Screenshot Analyzer

A Python application that captures screenshots and uses Google's Gemini AI to describe what's in the image.

## How to Run

```bash
uv run screenshot_app.py
```

## Usage

- Press **Enter** to take a screenshot and get an AI description
- Press **Esc** to exit the application

## Dependencies

- google-generativeai
- mss
- pynput
- Pillow

The dependencies are automatically installed when using `uv run`.
